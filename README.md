# Hermes数据库交互系统

## 项目介绍

本项目基于Socket和Websocket实现对外的连接，通过JSON数据包对指定数据库实现任意操作。
该项目核心为通过实现一个终端服务去操控不同种类的数据库，减少数据库交互方面的工作量

## 已支持数据库

- **MySQL**
